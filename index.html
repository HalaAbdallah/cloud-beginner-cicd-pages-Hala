What is CI/CD ??

CI/CD stands for Continuous Integration and Continuous Deployment. It is a method used in cloud systems to automate testing and deployment. When developers push code to a repository, the system automaticalliy checks it. After application or website is deployed automatically. This process makes development faster and more reliable.

My Pipeline

When I push my code to the main branch, Github Actions automatically runs a workflow. First it checks that required files exit. Then it uploads the website files. Finally it deploys the site to Github Pages automatically.
